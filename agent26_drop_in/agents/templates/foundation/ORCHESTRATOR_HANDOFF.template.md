## Orchestrator behavior on receiving this payload

1. Treat `references.context_registry` as authoritative project context — do **not** re-discover unless the registry is stale.
2. Apply `classification_hints` as starting hypotheses; refine during Phase 2 (Task Classification).
3. Activate every gate listed in `recommended_gates`.
4. Block any downstream agent whose required input is in `blocking_open_questions` until the question is resolved or converted into an explicit assumption.
5. Forward `assumptions_in_force` to every downstream agent that consumes them; require them to flag any that they cannot operate under safely.
6. Suggest `next_recommended_command` as the obvious next step.
